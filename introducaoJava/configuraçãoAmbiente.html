Configuração do ambiente Java

Neste conteúdo, você vai aprender:

  • o que é o ambiente Java e por que configurá-lo corretamente;
  • quais são as principais distribuições do JDK disponíveis;
  • como instalar o JDK no seu sistema;
  • como configurar a variável de ambiente PATH para usar os comandos Java;
  • como verificar se a instalação foi realizada com sucesso;
  • quais ferramentas facilitam a gestão e troca de versões do Java.

O que é o ambiente Java

O ambiente Java é o conjunto de ferramentas e configurações necessárias para escrever, compilar e executar programas Java no seu computador. Ele inclui o JDK (Java Development Kit) e a configuração correta do sistema para utilizar os comandos do Java.

Por que configurar o ambiente Java

Sem o JDK e o PATH configurados, o computador não reconhece os comandos Java, impedindo que você compile e execute seus programas. Um ambiente bem configurado é essencial para programar e desenvolver em Java com sucesso.

Distribuições principais do JDK

Oracle JDK

Versão oficial da Oracle, usada comercialmente, com suporte e atualizações. Algumas versões exigem licença para uso empresarial.

OpenJDK

Versão open source e gratuita do JDK, base para várias outras distribuições. É amplamente adotada pela comunidade.

Distribuições baseadas no OpenJDK

Entre as opções derivadas do OpenJDK, destacam-se:

  • Amazon Corretto — gratuita e com suporte da Amazon;
  • Adoptium, Azul Zulu, BellSoft Liberica, entre outras.

Todas são compatíveis para o desenvolvimento e execução de aplicações Java.

Etapas básicas para a configuração

  1. Instalar o JDK: escolha e baixe uma distribuição compatível com seu sistema operacional, como o Oracle JDK ou uma versão baseada no OpenJDK.
  2. Configurar a variável de ambiente PATH:
    • permite usar os comandos java e javac no terminal, em qualquer diretório;
    • essencial para compilar e executar seus programas Java com facilidade.
  3. Verificar a instalação: no terminal, digite os seguintes comandos:

    java -version
    javac -version

    Se tudo estiver correto, os comandos exibirão a versão instalada do Java.

Ferramentas que facilitam a gestão do Java

SDKMAN! (Linux e macOS)

O SDKMAN! é uma ferramenta que permite instalar, atualizar e alternar entre versões do Java (e outras SDKs) de forma simples, sem a necessidade de configurações manuais.

⚠️ Windows: o SDKMAN! não está disponível oficialmente para Windows, mas você pode usar gerenciadores como o Chocolatey para facilitar a instalação.
